1、需要提供一个配置文件命令为(lbm.json)
   (1)old指修改分页之前的配置，new指修改分页之后的配置
   (2)key是指根据哪一列、几列排序，多个列之间用逗号分割。分页前与分页后的排序字段名必须相同，否则影响对比结果
   (3)column指第二结果集的返回列的集合，需要从分页后的列集合中删除PAGE_TOTALNUM列
2、需要提供两个数据文件：修改分页之前的数据文件(命名为lbm_old.txt)和增加分页之后的数据文件(命名为：lbm_new.txt)
3、修改start.cmd，传入lbm参数

注意事项：
	1、以上的lbm均指具体的lbm编号，如：L1192162
